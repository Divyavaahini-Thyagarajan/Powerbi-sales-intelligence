# Sales Performance & Profitability Analytics Platform

## Overview
This project is an end-to-end Power BI analytics solution designed to deliver executive-ready insights into sales performance, profitability, product strategy, and customer value.

The solution follows enterprise BI best practices including star-schema modeling, advanced DAX calculations, and role-based dashboard design.

## Key Features
- Star schema data model
- Advanced DAX (YTD, MTD, YoY, Margin, AOV, ASP)
- Executive, Sales, Product, and Customer dashboards
- Drill-through to transaction-level details
- Export-ready detail page
- Comprehensive business and technical documentation

---

## Dashboard Pages
1. Executive Overview – Business health & growth
2. Sales Performance – Team and executive execution
3. Product Performance – Category & product profitability
4. Customer Analysis – Customer value & concentration
5. Detail / Export – Transaction-level drill-through

## Data Model
- Fact table: Sales transactions
- Dimensions: Date, Customer, Product, Team, Sales Executive, Segment
- Star schema with single-directional relationships
